# Quickstart

Get your Obsidian-powered wiki running quickly.

## 1. Install

```bash
npm install
```

## 2. Run the app

```bash
npm run dev
```

Open `http://localhost:3000`.

## 3. Optional: sync from Obsidian

```bash
export VAULT_PATH="/path/to/your/vault"
npm run sync
```

## 4. Check quality before shipping

```bash
npm run lint
npm run test
npm run typecheck
npm run build
```
